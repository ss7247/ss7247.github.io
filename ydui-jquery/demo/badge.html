Badge

1、说明

1.1、示例：http://m.ydui.org/html/badge.html

1.2、包含圆形样式和方形样式；

提供五中颜色样式：默认、badge-primary、badge-danger、badge-warning、badge-hollow；

2、调用

圆形角标
1 2 3 4 5

标签添加 .badge 和 相应色系class类。

<div>
    <span class="badge">1</span>
    <span class="badge badge-primary">2</span>
    <span class="badge badge-danger">3</span>
    <span class="badge badge-warning">4</span>
    <span class="badge badge-hollow">5</span>
</div>
方形角标
123 321 333 4444 55555

标签添加 .badge .badge-radius 和 相应色系class类。

<div>
    <span class="badge badge-radius">123</span>
    <span class="badge badge-radius badge-primary">321</span>
    <span class="badge badge-radius badge-danger">333</span>
    <span class="badge badge-radius badge-warning">4444</span>
    <span class="badge badge-radius badge-hollow">55555</span>
</div>
